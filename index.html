Myrtille

A native HTML4/5 Remote Desktop Protocol client

Welcome to Myrtille!

Myrtille is a simple yet efficient way to connect an RDP server through a web browser.

It requires only a simple browser (no plugin, extension, application or configuration whatsoever). Thus, it works straight from any web device (including touch pads and mobiles).

This project started back as a PoC in 2007, then became commercial in 2011 before going open source this year.

How does it works?

Myrtille is basically a web interface to a standard RDP client running on an HTTP gateway. It sends the user actions up to the RDP server then receives the resulting display.

By doing so, it acts like many other equivalent solutions (Guacamole or FreeRDP-WebConnect, to name only some).

However, it's one of the few available for .NET C#. It also supports both HTML5 and HTML4 (compatility with old browsers, starting from IE6 (!), or in case HTML5 websockets are blocked by a firewall or proxy).
